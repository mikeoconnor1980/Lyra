# Lyra - Product Specification

![Lyra Logo](images/lyra-logo.png)

Lyra is a project planning tool designed to handle maintenance of plans for single or multiple projects. It is flexible and configurable.

## Key Components

| Module | Description |
|--------|-------------|
| [Landing Page & Branding](01-landing-page.md) | Landing page design, colours, font, logo |
| [Overview & Key Principles](02-overview.md) | Product overview, key principles, and feature summary |
| [Plan Structure](03-plan-structure.md) | Hierarchical tree structure (Enterprise > Portfolio > Programme > Project > Task) |
| [Plan & Task Attributes](04-plan-task-attributes.md) | Task attributes, columns, status values, and plan screen layout |
| [Dependencies](05-dependencies.md) | Dependency management, dependency dialogue, and dependencies log |
| [Baselining](06-baselining.md) | Baseline setting, baseline history, and baseline numbering |
| [Reporting](07-reporting.md) | Report builder, report editing, saving, export, and canned reports |
| [Change Control](08-change-control.md) | Audit log, change management, approval thresholds, and change workflow |
| [AI Plan Generation](09-ai-plan-generation.md) | Automated plan generation from existing project documentation |
| [Platform & Requirements](10-platform-requirements.md) | Solution architecture, platform requirements, and pricing |
