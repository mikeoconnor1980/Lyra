# Lyra

![Lyra Logo](.agent-context/0-knowledge/images/lyra-logo.png)

Lyra is a flexible and configurable project planning tool designed to handle maintenance of plans for single or multiple projects. It provides comprehensive project management capabilities with a focus on flexibility, executive reporting, and AI-assisted planning.

## Key Features

- **Plan Maintenance**: Day-to-day running of plans using Gantt charts or task rows
- **Change Control**: Audit log and change management with user-set thresholds
- **Dependency Management**: Separate workspace for task relationships with reversible decisions
- **Reporting**: Flexible, configurable reports designed for boardroom presentations
- **AI Plan Generation**: Automated first drafts from existing project materials
- **Product Management**: Kanban view integration with plan tasks
- **Calendar View**: Visual task scheduling
- **Risk & Issue Register**: Automatic generation and AI-aggregated status commentary

## Key Principles

- **Flexibility**: Fully configurable by users
- **Executive Reporting**: Best-in-class boardroom-ready reports
- **Change Control**: Real workflow management
- **Dependencies**: Useful and practical dependency handling
- **AI Integration**: Intelligent plan generation and aggregation

## Architecture

- **Frontend**: Angular
- **Backend**: C# / .NET
- **Database**: SQLite (POC), Azure SQL (production)
- **Deployment**: Docker Compose (VPS), Azure Container Apps (cloud)
